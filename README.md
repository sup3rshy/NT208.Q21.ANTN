# SocialShield - Chrome Extension
Repo này cho môn học Lập trình ứng dụng Web - NT208.Q21.ANTN.
Mục đích ban đầu của project này là tạo ra một extension có chức năng như một Social Tracker: chụp snapshot list following hoặc follower instagram của một user bất kì, có chức năng so sánh snapshot ở các thời điểm khác nhau.
