Project information and credits

Project Overview

SocialShield
Social Media Security & Connection Monitor
Version 1.0.0

SocialShield is a Chrome extension designed to help users monitor and protect their social media accounts. It provides real-time connection tracking, privacy scanning, bot detection, link safety checking, and comprehensive security analysis for Instagram and Twitter/X platforms.

Key Features

📸
Connection Snapshots
Capture and track followers/following lists over time with automatic change detection
🔍
Privacy Scanning
Detect exposed personal information (PII), emails, phone numbers in profiles
🤖
Bot Detection
Identify suspicious bot-like accounts among your followers and following
🔗
Link Safety
Scan links on social media pages for phishing, malware, and other threats
🎭
Impersonation Detection
Find accounts that may be impersonating your identity
🛡️
Security Score
Comprehensive security assessment with actionable recommendations
